AI hỗ trợ học tiếng Pháp — 14.400 lượt hỏi mỗi ngày, không cần thẻ tín dụng.

Hướng dẫn lấy Groq API key (miễn phí, không cần thẻ)
  1. Truy cập Groq Console
    console.groq.com/keys
  2. Đăng nhập bằng Google / GitHub
    Không cần thẻ tín dụng, không cần verify gì cả.
  3. Nhấn "Create API Key"
    Đặt tên bất kỳ → Submit → copy key (dạng gsk_...).
  4. Dán key vào ô bên trên rồi nhấn Lưu
    Xong! AI dùng được ngay.
